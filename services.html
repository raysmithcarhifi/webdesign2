Content Management

We provide a full content management system (CMS) with every website design we create. From single page brochure sites through to bespoke multi page website’s, every aspect of content can be modified via password protected easy to use control panel.

Some features available with our content management system include:

  • The ability to update change and format all text, including font type, size, colour, paragraphing and alignment.
  • A complete file manager allowing you to insert, remove, resize and create onsite images.
  • Site linking functionality providing the ability to create links to other website’s as well as links to other pages within your own website.
  • The ability to upload file types (for example, PDF, Word document, Excel file etc) and offer it for download by your website visitors.
  • No software needed to download or install.
  • No ongoing fees or licences, the CMS system is yours included in the cost of the website.

Many more features are available and although full instructions are provided we offer initial hand holding guidance and help is available anytime. The CMS can be accessed from any internet connection allowing you to update your own website at all times.
